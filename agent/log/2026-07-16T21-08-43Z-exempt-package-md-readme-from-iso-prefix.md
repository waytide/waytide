# Exempted the two structural package files, `package.md` (the mechanical `include:` declaration) and `README.md` (the informative introduction), from the ISO-8601-UTC filename-prefix convention (`agent-file-names-iso8601`) in the Agent Norms composite repo. They are package-level structural files — a manifest and an intro — not dated agent artifacts, and they are the natural repo-root files of each component repo, so a datetime prefix would be noise. This is a deliberate amendment to the iso8601 rule; the **foundation** package's copy of that rule gets an explicit exemption note when foundation is migrated. Rule files under a package keep the full ISO prefix as before.
